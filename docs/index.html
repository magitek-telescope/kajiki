ドラッグアンドドロップによる快適なアップロードが可能

どんなクライアントよりも素早く、わかりやすく。

Kajikiは、あらゆるデータを何よりも素早く共有することを目的として設計されています。

ファイルシステムのパス構造を用いたインターフェースによって、GUIのファイルマネージャー特有の煩雑さから解き放たれながらも、ドラッグアンドドロップでのアップロード専用とすることで、視覚的なファイル選択を可能にしています。

高速かつセキュアなデータ転送

信頼と速度、両方を担保します。

Kajikiは外部に独自のサーバーを持たない設計となっているため、Amazon Web Service上の認証情報は、Amazon公式のシステムを介してのみ通信され、漏洩に関するリスクを最小限に抑えた状態でのデータ転送が可能です。

また、その性質により、ファイル転送を中継するサーバーがなく、高速で快適なアップロードが可能となっています。
